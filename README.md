# **KhoPhim101** *[NextJS, MongoDB]*

Website xem phim miễn phí không quảng cáo.

Link website : https://khophim101.vercel.app/

Nguồn API Phim : https://ophim1.cc/api-document

- HTML, CSS & JavaScript
- ***NextJS***
- TailwindCSS
- ***MongoDB***
